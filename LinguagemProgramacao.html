Linguagem de Programação

Ao longo das quatro primeiras semanas do semestre, a disciplina de Linguagem de Programação focou nos fundamentos de HTML, CSS, JavaScript, Lógica de Programação e GitHub. Durante esse período, desenvolvemos projetos práticos que permitiram a aplicação desses conceitos, construção e consolidação de habilidades na área de programação web.

Primeira Semana

O primeiro projeto consistiu na criação de um site de receitas de bolo. Essa atividade nos permitiu conhecer, explorar e praticar os fundamentos do HTML, como a criação de estrutura de páginas, a utilização de tags e a organização do conteúdo. Além disso, foram aplicados conceitos básicos de formatação, utilizando o próprio HTML, onde buscamos tornar a página visualmente mais agradável e estruturada. Ao final da semana, desenvolvemos a base necessária para entender o HTML semântico e CSS, assuntos que seriam abordados na próxima etapa.

Segunda Semana

O segundo projeto envolveu a criação de um currículo pessoal e uma carta apresentação em formato de site. Essa atividade nos permitiu aplicar os conhecimentos adquiridos, dentro e fora da sala de aula, na construção de uma página pessoal. Destacamos toda nossa bagagem profissional e acadêmica expondo nossas habilidades e competências, experiências e qualificações. Foram utilizados os fundamentos de HTML semântico para estruturar o conteúdo do currículo e a documentação do código, além de, desenvolver principalmente fundamentos de CSS para estilizar a página.

Terceira Semana

O terceiro projeto consistiu no desenvolvimento de um site que calcula os descontos na folha de pagamento. Nessa atividade, aplicamos também os conceitos de HTML e CSS para desenvolver a estrutura de um site com formulários de entrada de dados e utilizamos JavaScript para realizar os cálculos necessários. Esse foi o nosso primeiro contato direto com uma Linguagem de Programação e nos permitiu uma compreensão inicial à lógica de programação, tratamento de eventos e a manipulação de dados em JavaScript.

Quarta Semana

O último projeto abordou a criação de um site que recebe código HTML e JavaScript como entrada e, por meio da manipulação do DOM (Document Object Model) retorna uma página já formatada em CSS com o código de entrada. Essa atividade nos proporcionou uma abordagem prática na manipulação de strings, no processamento de código HTML e JavaScript e na aplicação de estilos CSS. Além disso, foi enfatizada a importância do controle de versão e colaboração utilizando o GitHub, onde pudemos compartilhar e versionar nossos projetos.
